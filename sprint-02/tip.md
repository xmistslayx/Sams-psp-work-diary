Add google translate for the additional languages

This issue was about adding google translate support for extra languages that were not already properly built into the system. This was important because NHS PathFinder is meant to help as many users as possible, especially in a hospital where not everyone will speak English confidently. It helped improve accessibility and made the app more inclusive for different patients and visitors.

As a patient, I want to see how many steps are left in my route so that I know how far I have to go.

This issue was about making the navigation feel clearer for the user by showing how many steps were left in the route. The reason this matters is because if someone is in a hospital and trying to find somewhere important they do not just want directions, they also want to know how much further they have got to go. It makes the route feel easier to follow and gives the user a better sense of progress.

As a patient, I want to see the department location on a map so that I can visualise where it is.

This issue was about adding a map view for the department location so the user can picture where they are heading instead of just reading text directions. This improves the user experience because some people understand things better visually. In a hospital setting this is useful because the buildings can be confusing and a map helps make the destination easier to understand.

Images avalable in managing paths page

This issue was about making images available in the managing paths page so that when paths are being managed the related images can also be seen. This makes the admin side of the system better because it is easier to check that the correct media is attached to the correct route. It also helps when reviewing or updating path information.
